# Small .NET Framework WinForms Applications for Practicing
- *This repository contains the source code for miscellaneous WinForms applications that are not worthy putting on the portfolio.*
- *You can see live demostrations for them on [this playlist](https://www.youtube.com/playlist?list=PLvj5wyoWpaub4R5gB6llZ-D1Id9ukAYnj).*

# How to open these projects on Visual Studio 2022
1. Press Ctrl+Shift+O (Open Project/Solution)
2. Select the `.csproj` file from the project files.

# How to add them into a solution
1. Open an existing solution.
2. From the Solution Explorer, right-click the solution name -> Add -> Existing Project
3. Select the `.csproj` file from the project files.
